Colab Notebook to PDF Converter

(created for 7.571)

Drop .ipynb file here or click to select

Or copy these commands into your terminal to convert locally:

# install required Python dependencies
pip install nbconvert
# install LaTeX + pandoc (macOS)
brew install --cask mactex-no-gui && brew install pandoc
# convert to PDF
jupyter nbconvert --to pdf your_notebook.ipynb
